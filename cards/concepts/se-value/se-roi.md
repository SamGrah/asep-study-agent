What return on investment (ROI) does adding SE effort yield?

---

Depends on SE already in place:
- A project using **no** SE activities: adding SE carries a **7:1 ROI** (each
  cost unit of SE reduces total cost by ~7 units).
- At the **median** level of projects surveyed: **3.5:1 ROI**.

(§1.2, p.6, Table 1.2)
